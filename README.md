# AnyDesk Legacy (v6.0.1) for Arch Linux

This is an AUR package for the stable legacy version of AnyDesk (v6.0.1).

## Why use this version?
- **No Nag Screens:** Avoids the 'Commercial Use Detected' timeouts found in v6.1+.
- **Stability:** Fixes freezing and Wayland issues present in newer versions.

## Installation
```bash
yay -S anydesk-legacy-bin
```
